# BAN 501: Predictive Modeling

## Course Companion

Welcome to the BAN 501 Course Companion. This resource provides comprehensive coverage of predictive modeling concepts, from foundational machine learning principles to advanced deep learning applications. It is the reading companion for the online, asynchronous offering of the course and is meant to be read alongside the module videos and hands-on notebooks.

## How to Use This Companion

- **Sequential reading**: Work through Modules 1-10 in order for a complete learning path
- **Reference**: Jump to specific topics using the navigation sidebar
- **Search**: Use the search bar to find specific concepts or terms
- **Deep dives**: Explore the appendices for extended, optional technical discussions that extend individual modules
- **Verified examples**: Every numerical example in this companion is produced by a verification script, so the numbers you read are reproducible

## Prerequisites

This companion assumes familiarity with:

- Python programming (at least one semester)
- Introductory statistics (mean, variance, distributions)
- Linear algebra fundamentals (matrices, vectors)

## Deep Dives (Appendices)

The appendices provide deeper, optional technical explorations. Each extends a specific module and is supplemental reading unless assigned:

- **Data Preparation & Feature Engineering** (extends Module 1): data quality, imputation, scaling, encoding, leakage
- **Model Evaluation & Validation** (extends Module 1): metrics, cross-validation, bias-variance
- **Handling Imbalanced Data** (extends Module 3): thresholds, class weights, SMOTE
- **Neural Networks as Universal Approximators** (extends Module 6): why networks can learn any function
- **Surprising Phenomena in Deep Learning** (extends Module 6): double descent, grokking, emergent abilities
- **CNN Architecture** (extends Module 7): convolution, weight sharing, receptive fields, parameter accounting
- **Transformer Architecture** (extends Module 8): self-attention step by step, multi-head reshape
- **Time Series Forecasting** (extends Module 10): decomposition, ARIMA, Prophet, walk-forward validation
